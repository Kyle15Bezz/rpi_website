Embedded Systems Project

👋 Hi everyone! I am Kyle and this Blog will aim to explain all steps followed in the completion of this project.

🚀 Problem Statement

Creating a weather display system that uses a Raspberry Pi, LCD, joystick, and speaker. Users can select their country and see the weather with all weather conditions on the LCD. The system will also speak the current weather temperature. The challenge is to combine hardware components, get real-time weather data from APIs, use the joystick to control the system, and display the weather on the LCD. The weather description would also be emailed to the user. The project aims to provide a fun, informative weather experience for users.

Interfacing LCD

The first step was to integrate raspberrypi with LCD as the lcd comes with i2c communication protocol by default so i needed to enable the I2C protocol from Raspberrypi by running sudo config command i also configure the SDA and SCL pins on Raspberrypi.

The Vcc and Ground pins were connected to the 5v and ground pins of raspberrypi respectively while the SDA and SCL pins were connected to GPIO 2 and GPIO 3 pins of raspberrypi respectively.

After the connections the next step was to test the lcd with some demo text to check whether the lcd is working perfectly or not. For this necessary library was installed and a github repository was cloned.

git clone https://github.com/the-raspberry-pi-guy/lcd

after cloning i cmd into the lcd folder. All the requirements were installed using the below command

sudo sh install.sh

After running the above command and rebooting the pi i ran the following script

python demo_lcd.py

The code ran successfully and i got the following output on lcd meaning the lcd was perfectly working.

🕹️ Interfacing the Joystick

After interfacing the LCD, the next step is to interface the Joystick. The Joystick comes with five pins: two for power and ground, and the remaining three for the x-axis, y-axis, and the button on the joystick.

The ground and +5v pins of joystick was connected directly with the ground and +5v pins of raspberrypi respectively. The VRX, VRY and SW pins were connected with GPIO 17, 27 and 22 pins of Raspberrypi respectively.

After the successful connections the next step was to write code and check the output for each action of joystick.

🌥️ Configuring Weather API

The next step is to get real time weather information that would be displayed on the LCD. For the following purpose i looked into some free weather APIs that were available and found out OpenWeatherMap API.

OpenWeather Ltd, provides global weather data via API, including current weather data, forecasts, nowcasts and historical weather data for any geographical location

I got the api key and targeted specific countries and cities in the browser to check if the API was working fine. The Weather API was working fine so i wrote the script in python to get the weather details of specific countries i got json response and from that response took out the necessary details that were to be shown o the LCD screen.

Following Info was extracted from Json response:

  1. Temperature
  1. Humidity
  1. Weather Description
  1. Wind Speed

Software integration with all Components

For the final step of this project, all the components were operated based on their functional requirements. A total of 10 countries and cities were added to the list, and with the help of a joystick, a specific country or city could be selected by moving forward. Moving downward would select the next city or country, and moving backward would select the previous option

The button on the joystick is used to select the specific country for sending a request to the openweathermap API. The user is prompted to enter the emial address where they want to receive weather email updates. The data is then retrieved and displayed on the screen. Additionally, the user receives an email notification when the data is received. A text to speech feature was added that converts the given text into the speech

pip install pyttsx3

The above library works offline, and is compatible with both Python 2 and 3. By using this library in speech the current weather of the city can be heard. For the sound output a speaker was attached with the 3.5mm jack on raspberrypi. An led was attached with the raspberrypi gpio pin the main aim of the led was to when the user is prompted to enter the email address the led is turned on when the email address is entered the led light is turned off.
